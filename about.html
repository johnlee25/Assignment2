Welcome To John Lee's Toronto Landmarks

About

This website was made by John Lee as part of the 4WW3 Fall 2019 project. This website allows users to register and log-in to their own accounts which will be setup in part 3. The home page displays the main page of the site which users can browser all content that is on the server. Clickling on a map marker will display more information regarding the landmark and allow users to jump to the landmark page Clicking on a object in the side bar will jump to that page for more information. Users can use the search page to search a landmark by location, name, or rating. By clicking the red location icon and allowing location services to run, your current location will be auto filled in. On the search results page, there is table at the bottom under the map showing you the searched results. Above that table, the map will display the markers for the objects that were returned on the search. Clicking on a marker will give you more information and allow you to navigate to the landmark page. Clicking on the name of the landmark from the table will also direct you to the landmarks page. On the users tab, hovering over it will show you two options, login or register. Login allows the user to sign into their pre existing account. Register allows the user to create a new account on the server. After the user is loged in, the user tab becomes signout.
